🛠️ One-time setup (then hands-free forever)

  1. Go to api-console.zoho.in → create a Self Client.
  2. Copy the Client ID & Client Secret.
  3. In the Self Client, generate a code with scope ZohoBooks.contacts.READ (valid 10 min).
  4. Exchange that code for a Refresh Token (one-time) — the refresh token never expires.
  5. Find your Organization ID in Zoho Books → Settings → Organisation.
  6. Paste all four here, pick region .in, Save → Test → Sync.

After this, the app auto-refreshes tokens and can re-sync any time — no further action needed. Until configured, vendor names come from the Books export + GSTIN cache automatically.
